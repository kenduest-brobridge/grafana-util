Change bundle JSON contract

Command:
  grafana-util change bundle --dashboard-export-dir ./dashboards/raw --alert-export-dir ./alerts/raw --output-file ./sync-source-bundle.json

kind:
  grafana-utils-sync-source-bundle

Top-level keys:
- kind
- schemaVersion
- toolVersion
- summary
- resources
- providers
- secretPlaceholders
- alertArtifacts
- resources[]
  - kind
  - identity
  - sourcePath

Notes:
- `change bundle` writes the contract to `--output-file`.
- It does not switch render modes with `--output-format`.
- Add `--also-stdout` only when you intentionally want both file output and stdout.
