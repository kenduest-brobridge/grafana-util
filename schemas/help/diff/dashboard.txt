Dashboard diff JSON schema

Command:
  grafana-util dashboard diff --url http://localhost:3000 --basic-user admin --basic-password admin --input-dir ./dashboards/raw --output-format json

General rule:
- Branch on `kind` plus `schemaVersion` before reading nested fields.
- `--output-format json` is the machine-readable path.
